Xadrez em C ♙

Por causa de uma disciplina escolar com a qual trabalhei com a linguagem C, me desafiei a criar um jogo de xadrez de dois jogadores, local (sem partidas online), no formato de terminal.

Esse jogo também foi publicado na Flavortown, uma plataforma de programação e desafio global voltada a adolescentes.

O jogo possui como recursos:

  1. Partida de xadrez local para duas pessoas;
  2. Sistema de salvamento de partidas;
  3. Uso de cores ANSI para colorir o tabuleiro e adicionar mensagens especiais;
  4. Sistema de verificação de jogadas fiel às regras oficiais do xadrez, incluindo movmentos especiais como roque, afogamento, etc.;
  5. E sistema de replay das partidas.
